# Code of Conduct

## Our Commitment

We want this project to be welcoming, respectful, and beginner-friendly for everyone.

## Expected Behavior

- Be respectful and constructive.
- Be patient with beginners and learners.
- Give feedback focused on code and behavior, not people.
- Use clear and professional language.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Discrimination of any kind.
- Sharing private information without consent.
- Repeated disruptive or hostile behavior.

## Enforcement

Maintainers may remove comments, reject contributions, or block users who violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private maintainer contact request through a GitHub issue and mark it as confidential in the title.
